## 1. 项目简介
本项目是一款基于机器学习（SVM）与自然语言处理（NLP）技术的智能评价分析系统，旨在解决电子商务（网购）行业中普遍存在的"刷单"、"虚假好评"及"恶意差评"问题。系统包含一个高精度的后端检测模型，以及一个具备现代化 UI/UX 设计的 Web 前端交互界面，实现了从数据清洗、特征提取、模型训练到实时检测的全流程闭环。

## 2. 核心创新点

### 2.1 多维特征融合策略 (Hybrid Feature Fusion)
- **打破传统**：不单纯依赖 TF-IDF 等词频统计。
- **融合创新**：结合了**统计行为特征**（如感叹号密度、数字占比、重复字符模式）与**语义内容特征**（情感词分布、商品关键词提及率），显著提升了对"机器刷单"和"模板化评价"的识别能力。

### 2.2 领域知识工程 (Domain-Specific Engineering)
针对网购电商场景构建了专属词库：
- **刷单特征词库**：识别"默认好评"、"返现"、"亲"、"老顾客"等高频刷单术语。
- **真实性验证词库**：检测"物流速度"、"包装完好"、"材质"、"做工"等真实用户关注的维度。

### 2.3 可解释性交互设计 (Explainable AI)
- **拒绝黑箱**：前端不仅输出真假结果，还可视化展示**判定依据**（如高亮特征词、置信度条、负面/正面词汇统计），增强用户信任。

### 2.4 现代化架构
- **前后端分离**：前端采用玻璃拟态 (Glassmorphism) 设计，后端提供标准 REST API。
- **即时响应**：优化的轻量级模型架构，确保毫秒级的检测速度。

## 3. 技术实现

### 3.1 核心算法层 (Python/Scikit-learn)
- **模型**：SVM (支持向量机) + RBF 核函数。
- **特征工程**：
  - TF-IDF N-gram (1-2元) 向量化。
  - 手工构造的 8 维统计特征向量。
  - 特征空间融合 (`scipy.sparse.hstack`)。
- **数据处理**：完整的 NLP 流水线（清洗、分词、去停用词）。

### 3.2 服务层 (Flask)
- 提供 RESTful API (`/predict`)。
- 支持模型持久化 (Persistence) 与自动热加载。
- 跨域资源共享 (CORS) 支持前后端独立部署。

### 3.3 前端交互层 (HTML5/CSS3/JS)
- **设计**：原生 CSS 实现 Glassmorphism 风格。
- **逻辑**：异步 API 调用，动态 DOM 操作，丰富的动画交互。

## 4. 优化记录 (Optimization Log)

### 已完成优化
1.  **架构升级**：
    - 将原有的前端纯逻辑模拟升级为**真实的前后端分离架构**。
    - 引入 Flask 搭建 API 服务器，使前端能够调用 Python 训练的真实 SVM 模型。
2.  **模型工程化**：
    - 实现了模型的保存 (`save_model`) 和加载 (`load_model`) 机制，避免每次启动重复训练。
    - 增加了 API 的健壮性处理（健康检查、错误捕获）。
3.  **依赖管理**：
    - 更新 `requirements.txt`，补充了 Web 服务所需的依赖库。

### 后续优化方向
- **模型演进**：引入 BERT/LSTM 等深度学习模型以捕捉更深层的语义信息（如反话识别）。
- **数据增强**：利用 Word2Vec 扩充特征词库，提升模型泛化能力。
- **可视化报表**：增加数据集分布的图表展示功能。

## 5. 快速开始

### 环境依赖
```bash
pip install -r code/requirements.txt
```

### 启动服务
1. **启动后端 API**：
   ```bash
   python code/server.py
   ```
   *服务将运行在 http://127.0.0.1:5000*

2. **启动前端**：
   直接浏览器打开 `frontend/index.html` 即可。
