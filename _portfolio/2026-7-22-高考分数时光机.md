- **在线体验**: [高考分数时光机 →](https://lyzbcy.github.io/gaokao-time-machine/#/)

## 这是什么

「高考分数时光机」——一个互动测试题网站。回答几道趣味问题，测出你的高考分数放到 2026 年能上什么大学。

## 玩法

1. 进入网站，回答一系列关于学习习惯、考试心态的趣味选择题
2. 系统根据答案"计算"你的高考分数
3. 展示这个分数在 2026 年能上的大学层级（带夸张的文案和表情）
4. 生成分享卡片，可以发给朋友 PK

## 技术实现

用 **lyzbcy-测试题制作 skill** 生成。核心是：
- 题目数据驱动（题目/选项/分值写在 JSON 里，改题不用改代码）
- 「开盲盒式」结果展示（答完立刻抽到带稀有度的称号）
- 云排行榜（朋友间 PK）
- 分享卡片生成
- 捞鱼个人品牌入口（表情包种草 + 赞赏码）

部署在 GitHub Pages，纯静态，免费托管。
