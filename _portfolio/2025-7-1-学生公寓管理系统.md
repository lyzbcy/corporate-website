2025年7月1日

## 🚀 项目简介
本项目是一个基于 C/S 架构的学生公寓管理系统，旨在为学校宿管部门提供高效、便捷的寝室管理解决方案。系统涵盖了从学生入住、寝室分配到资产管理、出入登记的全流程业务。

## 🛠️ 技术栈

本项目采用了经典的前后端分离（C/S）架构，重点涉及以下技术：

### 1. 数据库层
- **核心数据库**: **MySQL** (Relational Database Management System)
  - 使用 InnoDB 存储引擎，确保数据的完整性和事务支持。
  - `utf8mb4` 字符集支持，全面兼容多语言及特殊字符。
- **高级特性应用**:
  - **存储过程**: 核心业务逻辑（如`sp_allocate_dorm`分配寝室、`sp_add_student`添加学生）均封装在数据库层，提高了执行效率和安全性。
  - **触发器**: 用于自动维护数据一致性（如人数变更自动更新寝室状态）。
  - **外键约束**: 严格的数据完整性约束，确保 `student`、`dorm`、`building` 等表之间的数据关联正确。
  - **索引优化**: 针对常用查询字段（如 `name`, `department`, `grade`）建立了索引，加速查询响应。

### 2. 客户端应用
- **运行平台**: Windows (Win32/Win64)
- **应用类型**: Windows 桌面应用程序 (`.exe`)
- **交互方式**: 图形用户界面 (GUI)
