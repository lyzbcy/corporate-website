这是一个基于 Python Tkinter 开发的模拟智能家居控制系统。该系统提供了一个图形化界面，用于管理和控制家庭中的智能设备，支持多用户管理、设备共享以及基于温度、湿度和时间的自动化控制功能。

### 📅 项目时间
- **完成时间**: 2024年11月25日

### 🛠️ 技术栈
*   **语言**: Python
*   **GUI 框架**: Tkinter (标准库)
*   **数据存储**: JSON
*   **其他**: `datetime` 模块

## 🤕 遇到的困难
当时是第一次尝试图形化界面，虽然这个不用装啥环境，但是改起来也有些难度，不像之前命令行界面那么直接。解决办法就是慢慢磨..
