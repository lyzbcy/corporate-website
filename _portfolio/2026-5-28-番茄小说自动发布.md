## 📊 项目信息
- **首次提交**: 2026年5月
- **项目类型**: 内容发布自动化工具
- **核心技术**: Python, 浏览器自动化, 番茄小说平台 API
- **项目开源**: [GitHub →](https://github.com/lyzbcy/fanqie-publisher)

## 这是什么

通过浏览器自动化实现番茄小说短故事的自动创建与发布。让 AI 写完小说后，自动完成登录、创建作品、填写信息、发布上线的全流程，不用手动复制粘贴。

## 核心功能

- **自动登录**：首次配置后保持登录态
- **自动创建作品**：自动填写标题、简介、分类等元信息
- **自动发布**：将 AI 生成的小说内容发布到番茄小说平台
- **批量处理**：支持一次发布多个短故事

## 使用方式

把仓库发给 AI 助手（小龙虾等），让它根据 README 部署即可。
