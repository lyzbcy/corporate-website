这是一个基于 Python 开发的模拟银行管理系统，旨在模拟真实的 ATM 业务流程。系统提供了用户开户、查询、存取款、转账以及管理员管理等一系列核心功能，并采用简单的文件存储方式实现数据持久化。

### 📅 项目时间
- **完成时间**: 2024年11月14日

### 🚀 项目概述

该项目完全使用 Python 语言编写，通过命令行界面 (CLI) 与用户进行交互。它模拟了完整的银行后台与前台交互逻辑，包括管理员权限控制、用户账户安全验证（如密码强度检测、身份证/手机号格式校验）以及资金流转处理。

### 🛠️ 技术栈

本项目主要采用了以下技术和库：

*   **编程语言**: Python 3
*   **核心模块**:
    *   `random`: 用于生成唯一的 6 位随机银行卡号。
    *   `re`: 用于实现复杂的格式校验（身份证号、手机号）。
    *   `time`: 用于模拟系统处理延时，优化用户体验。
    *   `pickle` / `os`: 用于数据的持久化存储与解析。
